Historias de usuario ordenadas de mayor a menor prioridad:

1. Como ciudadano quiero registrar una incidencia para dejar evidencia del caso y que pueda 
ser atendida por las autoridades correspondientes.

2. Como coordinador quiero recibir una alerta en Telegram para revisar rapidamente un 
nuevo reporte sin necesidad de estar en el sistema todo el tiempo.

3. Como miembro del equipo quiero consultar todos los reportes registrados para hacer 
seguimiento del estado de cada incidencia.

4. Como ciudadano quiero recibir confirmacion de que mi reporte fue recibido para saber 
que mi solicitud esta siendo procesada.

5. Como coordinador quiero filtrar reportes por tipo de incidencia para priorizar la 
atencion segun la gravedad del caso.
